# 📖 Overview

The **CSE Smart Quiz Portal** is an interactive web-based quiz application designed specifically for Computer Science Engineering students.

The system allows students to:

- Select their academic year
- Choose a subject based on their year
- Attempt multiple-choice questions
- View their final score
- Calculate percentage automatically
- Receive a grade based on performance

The project provides a simple and engaging way to practice technical concepts across different CSE subjects.

---

# ✨ Features

- Student Information Form

- Year-wise Subject Selection

- Multiple Choice Questions (MCQs)

- 10 Questions per Subject

- Automatic Score Calculation

- Percentage Calculation

- Grade Evaluation

- Progress Tracking

- Responsive User Interface

- Restart Quiz Option
